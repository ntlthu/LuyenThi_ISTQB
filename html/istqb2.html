4. Which of the following statements is a valid objective for testing?

A. To determine whether enough component tests were executed within system testing

B. To find as many failures as possible so that defects can be identified and corrected

5. Which of the following statements correctly describes the difference between testing and debugging?

5. Phát biểu nào dưới đây mô tả đúng sự khác nhau giữa kiểm thử (testing) và gỡ lỗi (debugging)?

A. Testing identifies the source of defects; debugging analyzes the defects and proposes prevention activities

A. Kiểm thử xác định nguồn gốc của defect; gỡ lỗi phân tích defect và đề xuất các hoạt động phòng ngừa

B. Testing shows failures caused by defects; debugging finds, analyzes, and removes the causes of failures in the software

B. Kiểm thử chỉ ra các failure do defect gây ra; gỡ lỗi tìm, phân tích và loại bỏ nguyên nhân gây ra failure trong phần mềm

C. Testing removes faults; debugging identifies the causes of failures

C. Kiểm thử loại bỏ fault; gỡ lỗi xác định nguyên nhân của failure

D. Testing prevents the causes of failures; debugging removes the failures

D. Kiểm thử ngăn chặn nguyên nhân gây ra failure; gỡ lỗi loại bỏ failure
